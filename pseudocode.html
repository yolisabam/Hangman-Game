Hangman Game Pseudocode

1. User chooses a letter from A to Z

2. Computer chooses a random value (current word) in a list of Beyonce's top hits

3. We determine if the user wins or not by

  • If user picks a letter and it is not in the current word, then the number of guesses decreases by 1

  • If user picks a letter and it is not in the current word, the letter shows up on the screen in a section that's titles "Letter's already used". If the letter pops up in this section, it has no effect anymore and cannot be used for the rest of the game.

  • If the user guesses a letter that is in the current word then that letter replaces the dashes where it would be

  • If the user guessed the entire word, then a new title appears on top of the game and a new image pops up in the column next to it.

4. We then add to their score

  • If user wins then we add one to their wins.

5. The computer then picks a new value from the list of beyonce's top hits and starts the game over
